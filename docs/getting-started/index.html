Emberのインストール 編集

Emberを始めるのは簡単です。 EmberのプロジェクトはコマンドラインビルドツールであるEmber CLIによって作成され管理されます。 Ember CLIは以下のものを提供します。

  • モダンなアプリケーションアセット管理(結合、縮小、バージョニング)
  • コンポーネントやルートなどの作成を手助けするジェネレータ
  • Babelを使ったES2015とES6のサポートと、本ガイド内で使われているJavaScript modules
  • QUnitを完全に活用するためのテストツール群
  • Emberアドオンのサポート

必要なもの

Git

Emberはたくさんの依存関係を管理するためにGitを必要とします。 GitはMacとほとんどのLinuxディストリビューションにはインストールされています。 WindowsユーザはこちらからGitインストーラをダウンロードできます。

Node.jsとnpm

Ember CLIはJavaScriptで作られており、最新のNode.jsのLTSを必要とします。 さらに依存しているものをnpmで取得します。 npmはNode.jsに同封されているため、Node.jsが入っていれば準備完了です。

もし、Node.jsがインストールされているか、正しいバージョンなのかが分からない場合は、以下のコマンドを実行して確認してください。

1
2

node --version
npm --version

もし、コマンドが見つからないといったエラーが発生したり、古いバージョンの場合は、

  • WindowsかMacの方は、Node.jsインストーラをダウンロードして実行してください
  • MacユーザはHomebrewを使ってNode.jsをインストールする人も多いです。Homebrewインストール後にbrew install node を実行し、Node.jsをインストールしてください。その他の手段では、Node.jsのサイトのダウンロードページから直接インストーラパッケージがあります。
  • LinuxユーザはNode.jsのサイトのダウンロードページを参照してください。

もし古いバージョンのnpmだった場合は、npm install -g npmを実行してください。

Watchman (任意)

MacとLinuxでは、Watchmanをインストールすることでファイルの監視のパフォーマンスを改善することができます。

インストール方法

npmを使ってEmberをインストールします。

1

npm install -g ember-cli

インストールが成功したか検証するには、以下のコマンドを実行してください。

1

ember -v

バージョンが表示されていれば準備完了です。
